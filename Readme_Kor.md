# 자바 콘솔 기반 키오스크

## 프로젝트 개요
이 프로젝트는 Java로 구현된 키오스크 시스템이며, 여러 단계(Lv)로 구성되어 있습니다.

사용자 이름 => admin 충분한 돈과 포인트가 있는 사용자로 프로그램의 버그 픽스 및 기능 확인을 위함

<br>

#### Lv1: 콘솔에 메뉴 출력
#### Lv2: `MenuItem.java` 클래스를 활용한 음식 항목 관리
#### Lv3: `Kiosk.java` 클래스에서 UI 및 결제 로직 처리
#### Lv4: `Menu.java` 클래스를 활용한 메뉴 카테고리 관리
#### Lv5: 장바구니 및 구매 기능 구현, `Lambda & Stream`을 활용한 장바구니 관리, `Enum`을 이용한 할인 적용, 신규 이용자 자동 등록 및 포인트 적립/사용 처리

### 요약
Lv5에서 고객은 메뉴에서 항목을 선택하고 장바구니에 추가한 후, 할인 옵션을 선택하여 주문을 진행할 수 있습니다.
신규 사용자의 이름, 잔액, 포인트는 자동으로 저장되며, 사용된 금액과 포인트 정보는 별도의 파일에 영구적으로 저장됩니다.

### 향후 개선 사항
1. 포인트 사용 및 적립 내역 저장
2. 구매 내역 저장 기능 추가
3. 신용카드 할부 결제 기능 구현
4. 직불카드, 현금, 신용카드에 대한 결제 로직 분리
5. UI 개선

## 기술 스택
- **언어**: Java 17+
- **빌드 시스템**: Maven or Gradle
- **IDE**: IntelliJ IDEA
- **버전 관리**: Git

## 주요 기능
### 고객 기능
- 메뉴 탐색 및 선택
- 장바구니에 항목 추가 및 주문 진행
- 신규 이용자 자동 등록
- 포인트 적립 및 사용

### 관리자 기능
- 메뉴 항목 추가, 수정 및 삭제
- 할인 정책 적용

---------------

## 실행 방법
각 패키지의 `Main.java` 파일을 실행하거나, `src/Main.java` 를 실행
